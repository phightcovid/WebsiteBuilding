K-12 School Closures and Child Care Operating Status Changes

Restrictions/Closures of State Parks including campgrounds, RV parks, playgrounds, main walking paths, education centers, restrooms, etc.


The interactive map below displays state park restrictions and closures through time.

State parks were considered as having restrictions or being closed if a state order was issued to entirely close a state park or other outdoor amenities within a state park.

Click on a state, or the circles displaying the new daily cases per 500,000 people for more information.
