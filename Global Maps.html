Ohio Case Data by Age Group Time Series

The graph below shows the total COVID-19 cases and % of daily cases by age in Ohio from March 2020 to July 2021. The y-axis as labeled on the left side of the graph corresponds to the colored columns. Each of these colored columns represents the percentage of new daily cases for a particular age group. Since the percentage of new daily cases for each of these age groups was calaculated using the total number of new daily cases out of the entire population (the denominator), the age groups can be accurately compared between each other. The y-axis on the right side of the graph corresponds to the black line showing the total number of new daily cases out of the entire population (this is the value of the denominator for calculating percentage of new daily cases for each of the age groups). The reported proportion of the B117 variant is shown as the grey line.
